# Liza Frontend

React marketplace frontend with authentication and product management.

## Quick Start

```bash
npm install
npm run dev
```

App runs on `http://localhost:3000` (backend expected at `http://localhost:8000`)

## Features

- Product listing and creation
- Order management
- User authentication (login/signup)
- Profile page with user products/orders

## Tech Stack

React 18, React Router, Tailwind CSS, Axios, Vite
